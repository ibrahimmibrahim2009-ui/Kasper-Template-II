# Kasper | Template II
### An experimental website for testing different design layouts and beautiful interfaces.
## Responsivity
### The design is fully responsive as it seems well organized at different screen sizes: small, medium, or large screens.
## What I learnt in my Journey?
1. I learnt how to change the list pullets with more attractive icons or even any beautiful shape instead.
2. I learnt how to change the formatting of quotaion marks in Quote element using "quotes" property and "open/close quote" values in Pseudo elements.
3. I realized the importance of testing the responsivity section by section, not after ending the whole website as it will lead to more useless effort.
